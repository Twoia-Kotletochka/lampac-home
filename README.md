# Lampac Home

Готовый к запуску **self-hosted backend Lampac NextGen** для медиаплеера **Lampa** — в одном
Docker-контейнере. Разворачивается одной командой и сразу даёт:

- 🎬 **онлайн-источники** — 70+ балансеров фильмов и сериалов;
- 🧲 **торренты** — встроенный **TorrServer** + торрент-парсер (**JacRed**, Jackett-совместимый API);
- 📺 **DLNA**, синхронизация профилей/закладок/таймкодов между устройствами;
- 🔓 **разблокировка** «Торрентов»/«Парсера» на магазинных сборках Lampa (обходит демо-режим Samsung/LG);
- 🇺🇦 **украинский IPTV** — свой плагин с сотнями каналов и авто-обновлением плейлиста;
- 🧹 отключённый навязчивый «shots» и подобный мусор на главной.

Весь трафик идёт через ваш сервер — часть блокировок обходится, публичные адреса не светятся.

## Быстрый старт

```bash
tar -xzf lampac-home-*.tar.gz     # если ставите из архива с образом
cd lampac-home
./install.sh
```

Или без архива (образ подтянется из реестра):

```bash
git clone <этот-репозиторий> lampac-home
cd lampac-home
./install.sh
```

После установки скрипт покажет адрес. Обычно это:

| Что | Адрес |
|-----|-------|
| Веб-Lampa | `http://SERVER_IP:9118` |
| Плагин для Lampa на ТВ | Настройки → Расширения → `http://SERVER_IP:9118/on.js` |
| Парсер торрентов | Jackett — настраивается автоматически |
| TorrServer | `http://SERVER_IP:9118/ts` |

На телевизоре вводится **только один** адрес — `…/on.js`. Всё остальное (источники, TorrServer,
парсер, IPTV, разблокировка) приезжает автоматически.

> Адрес сервера **нигде не зашит**: Lampac подставляет его сам через переменную `{localhost}`
> по тому адресу, по которому к нему обращается клиент. Поэтому конфиг работает на любом IP/домене
> без правок — просто используйте свой адрес там, где добавляете плагин `on.js`.

## Требования

- Docker + docker compose (плагин v2)
- Linux-хост (проверено на Debian). `network_mode: host` — нужен для DLNA/TorrServer.
- ~3 ГБ под образ; место под кэш/торренты — папка `storage/`.

## Что внутри

```
docker-compose.yaml          переносимый (все пути относительные)
install.sh                   установка одной командой
lampac-image.tar(.gz)        Docker-образ (только в самодостаточном архиве; в git его нет — тянется из ghcr.io)
lampac-docker/
  config/init.conf           конфиг Lampac (модули, плагины, парсер) — без IP, через {localhost}
  config/passwd.example      пример; реальный passwd создаёт install.sh и он в .gitignore
  plugins/unlock.js          снятие демо-режима (возврат торрентов/парсера в магазинных сборках)
  plugins/uatv.js            плагин украинского IPTV (каналы встроены), меню «Украïна ТВ» / «+»
  plugins/lampainit.js       парсер настраивается при каждом запуске
  plugins/on.js              инъекция парсера + фильтр shots + загрузка IPTV
  gen_uatv.py                генератор uatv.js из m3u-плейлистов
  refresh_uatv.sh            ежедневное обновление IPTV (ставится в cron)
storage/                     кэш и данные (создаются при установке)
```

## Обслуживание

- Обновить образ: `docker compose pull && docker compose up -d`
- Логи: `docker logs lampac`
- Обновить IPTV вручную: `./lampac-docker/refresh_uatv.sh`

## Оговорка

Все онлайн-источники и плагины сторонние — ставятся на свой риск, разработчики Lampa за них
не отвечают. Проект для личного использования.
