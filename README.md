# Lampac Home — self-hosted Lampac NextGen

Готовая сборка самохостируемого backend **Lampac NextGen** для Lampa: онлайн-источники,
встроенный TorrServer, торрент-парсер (JacRed/Jackett), DLNA, синхронизация — плюс наши
доработки: возврат торрентов/парсера на магазинных сборках Lampa, отключение shots,
и собственный плагин украинского IPTV.

## Быстрая установка

```bash
tar -xzf lampac-home-*.tar.gz
cd lampac-home
./install.sh
```

Если разворачиваете на сервере с другим IP:

```bash
SERVER_IP=192.168.1.50 ./install.sh
```

После установки:

| Что | Адрес |
|-----|-------|
| Веб-Lampa | `http://ВАШ_IP:9118` |
| Плагин для Lampa на ТВ | Настройки → Расширения → `http://ВАШ_IP:9118/on.js` |
| Парсер торрентов | Jackett, `http://ВАШ_IP:9118`, ключ пустой (настраивается сам) |
| TorrServer | `http://ВАШ_IP:9118/ts` |

На телевизоре достаточно добавить один плагин `.../on.js` — всё остальное (источники,
TorrServer, парсер, украинский IPTV, разблокировка) приезжает автоматически.

## Требования

- Docker + docker compose (плагин v2)
- Linux-хост (проверено на Debian 13). `network_mode: host` — нужен для DLNA/TorrServer.
- ~3 ГБ под образ, место под кэш/торренты (папка `storage/`).

## Что внутри

```
docker-compose.yaml          переносимый (все пути относительные)
install.sh                   установка одной командой
lampac-image.tar(.gz)        Docker-образ (только в самодостаточном архиве; в git-репо его нет — тянется из ghcr.io)
lampac-docker/
  config/init.conf           конфиг Lampac (модули, плагины, парсер)
  config/passwd              пароль root (генерируется при установке, если нет)
  plugins/unlock.js          разблокировка торрентов/парсера на магазинных сборках Lampa (демо-режим)
  plugins/uatv.js            плагин «Украïна ТВ» + «Украïна ТВ+» (каналы встроены)
  plugins/lampainit.js       правленый: парсер настраивается при КАЖДОМ запуске
  plugins/on.js              правленый: фильтр shots + загрузка IPTV + инъекция парсера
  gen_uatv.py                генератор uatv.js из m3u-плейлистов
  refresh_uatv.sh            ежедневное обновление украинского IPTV (ставится в cron)
storage/                     кэш и данные (создаются при установке)
```

## Обслуживание

- Обновить образ: `docker compose pull && docker compose up -d`
- Логи: `docker logs lampac`
- Обновить украинский IPTV вручную: `./lampac-docker/refresh_uatv.sh`

## Заметки

- Адрес `192.168.0.92` зашит в `on.js`, `uatv.js`, `init.conf`. При переносе на другой IP
  используйте `SERVER_IP=...  ./install.sh` (заменит автоматически).
- TorrServer при первом старте качает бинарник и стартует с RAM-кэшем; настройку дискового
  кэша (на `storage/data/ts`) при желании выставьте в его веб-UI `/ts` → Настройки.
- Все плагины/источники сторонние, ставятся на свой риск.
